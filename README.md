# AnaNoSleep & NankaiGayming

这是一个基于现代Vue3架构构建的网站。
- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Pinia** - 现代状态管理库
- **Sass** - CSS预处理器

## 功能特性
-  响应式设计（移动端适配）
-  明暗主题切换
-  现代化UI
-  组件化架构
-  状态管理
-  路由导航


## 开发命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
